# Changelog

## v0.1.0
- Initial working release.
- Quarto extension with `svg-frag` shortcode.
- Maps Inkscape layers to Reveal.js fragments.
- Responsive SVG CSS contribution (default behaviour).
- Parameters: `file` (required), `start` (default 0), `class` (default `fragment`), `width` (optional), `height` (optional).
- Examples and documentation.
